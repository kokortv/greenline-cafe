Категории меню

Структура: Основные категории (Завтрак, Обед, Ужин, Бар) → Подкатегории (Первые блюда, Горячее и т.п.) → Блюда. Подкатегория привязана к основной категории. Блюдо привязано к подкатегории.

QR-код меню для посетителей

Сгенерируйте QR-код для публичного меню. Посетитель наведёт камеру телефона и увидит меню с ценами — без возможности сделать заказ. Отметьте галочками категории, которые должны отображаться.

Опасная зона

Сброс данных — удалит все заказы, но сохранит настройки, категории и меню.

Очистка сессий

При каждом входе официанта/повара/админа создаётся сессия (токен), которая хранится в листе Settings под ключами session_.... Сессии живут 12 часов, но устаревшие записи могут накапливаться. Нажмите, чтобы удалить все просроченные сессии.

Новое блюдо

Будет показано повару под основным названием. Оставьте пустым, если перевода нет.

Введите значения в любые два поля — третье рассчитается автоматически. Наценка указывается в процентах (например, 50 = +50% к себестоимости).

Добавьте несколько вариантов товара (напр. "Сухарики: с сыром / с беконом / с солью"). Каждый вариант имеет свою цену. Перетаскивайте за значок ☰ для изменения порядка.

Снимите галочку для готовых товаров (вода, хлеб, соки, алкоголь) — они не появятся у повара. Порядок товаров меняется перетаскиванием в списке.
